Key Measurements
Main Screen
Enso circle: 260x260pt canvas
Circle radius: ~37% of canvas width
Circle offset: -30pt from center (shifted up)
Theme button: top 58pt, right 24pt, 28x28pt
Progress text: 32pt below enso
Swipe cue: bottom 36pt, line 28x1pt
Seal: 52x52pt circle, 2pt border
Toast: top 68pt, centered
Stats Screen
Horizontal padding: 28pt
Header: sticky, top padding 56pt
Stat blocks: 44pt bottom margin
Progress bar: 3pt height, 16pt top margin
Weekly chart: 100pt height, 6pt gap
Conditions grid: 3-col, 1pt gap, 16pt radius
Condition cell: 18pt vertical padding
Smart note: 2pt left border, 14pt padding
Settings Screen
Horizontal padding: 28pt
Group bottom margin: 32pt
Row padding: 16pt vertical
Row border: 1pt bottom, ink-whisper
Stepper button: 30x30pt circle, 1pt border
Stepper gap: 14pt
Toggle: 42x24pt, rail 24pt radius
Toggle knob: 18x18pt, 3pt inset
Reset button: full width, 16pt padding, 16pt radius
Enso Brush Stroke
Start angle: -0.55*PI (upper-left)
Min arc (0%): 12% of full circle
Max arc (100%): 94% of full circle
Base stroke width: 5pt (empty) to 23pt (full)
Alpha range: 0.06 (ghost) to 0.94 (bold)
Pressure profile: thick attack, thin sustain, press end, dry trail
Wobble: sin(t*53.7)*1.8 + cos(t*37.3)*1.1 + noise*1.4
Splatter: progress*12 dots, random offset along stroke
Animation Specs
Transitions
Stats open: translateY(100% -> 0), 0.7s, ease-brush
Main away: translateY(0 -> -40%) + scale(0.95), 0.7s + 0.5s opacity
Settings open: translateX(100% -> 0), 0.55s, ease-brush
ease-brush: cubic-bezier(0.22, 1, 0.36, 1)
ease-ink: cubic-bezier(0.33, 0, 0.2, 1)
Interactions
Ink ripple: scale(0 -> 4), opacity(0.7 -> 0), 1s ease-brush
Enso progress: lerp 0.07 per frame toward target
Seal stamp: scale(0 -> 1) + rotate(-15 -> -8deg), 0.6s
Ink wash: radial gradient fade in/out, 2.2s
Toast: translateY(-12 -> 0) + opacity, 0.5s
Swipe cue breathe: opacity 0.4 -> 0.8, 4s infinite
Sync dot pulse: opacity 0.5 -> 1, 3s infinite
